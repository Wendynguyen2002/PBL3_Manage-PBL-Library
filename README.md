Đọc kỹ hướng dẫn trong file này: https://docs.docker.com/desktop/setup/install/windows-install/ để tải Docker Compose về Win

Sau đó clone dự án về, mở terminal, cd PblManagement, đảm bảo dòng trong terminal như này ~/Desktop/repos/PBL3_Manage-PBL-Library/

Chạy 1 lần: docker compose down, docker compose build --no-cache, docker compose up

Sau đó mở Google và nhập link http://localhost

Để dừng chạy Docker thì nhấn Ctrl + C
